Overview

Accounts

subscription or API
An account is a named way to call a model: your Claude/Codex login (no key), an API key (link a credential below), or a local LLM (Ollama / LM Studio — give its endpoint). Workstreams pick one under “runs as”.

Credentials

macOS Keychain
Subscriptions are the default and need nothing here. API keys are optional. Secrets live in the macOS Keychain, never in a file, never shown again — choose Always Allow on the first Keychain prompt.
