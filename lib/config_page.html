Accounts

named auth instances · subscription or API
An account is a named instance an agent calls from — your Claude or Codex subscription (no key needed), or an API key. API accounts link to a credential below.

Credentials

API keys · macOS Keychain · never leaves this machine
Subscriptions (your Claude / Codex logins) are the default and need nothing here. API keys are optional — add one only if a role should run on API credit instead. Secrets are stored in your macOS Keychain, never in a file, never shown again after saving. On your first save macOS asks to allow Keychain access — choose Always Allow so later saves don't prompt.
